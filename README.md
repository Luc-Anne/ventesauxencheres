# Projet Java Full Stack

Site web d'enchères, basé sur un travail de groupe au cours de la formation à l'ENI durant 2022-23 et qui montre en partie ce que je sais faire en Java 11 SE et Java EE.

Nous avons pu faire l'essentiel du travail dans le temps imparti ; néanmoins, je continue ce projet en autonomie afin de pouvoir parfaire mes compétences en programmation d'un site web en java. J'ai notamment décider de repartir de zéro, en utilisant MAVEN, avec un nouveau BO plus complet et une base de données MariaDB au lieu de SQL SERVER.

Travail en cours, ne pas utiliser en production.
